\section{Matlab Quellcode}

\subsection{Was wird benötigt}
Für Listings (Quellcode-Fenster) benötigt man lediglich das Package:\\
\begin{lstlisting}
\usepackage{listings}
\end{lstlisting}
Möchte man den Quellcode einfärben, benötigt man zusätzlich:\\
\begin{lstlisting}
\usepackage{color}
\end{lstlisting}
